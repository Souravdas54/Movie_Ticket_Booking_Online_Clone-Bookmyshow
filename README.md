# 🎬 Movie Ticket Booking Online Clone (BookMyShow Clone)

A full-stack Movie Ticket Booking Platform inspired by BookMyShow, built using Next.js, TypeScript, Node.js, Express.js, MongoDB, and modern web technologies.

This application allows users to browse movies, select theaters and showtimes, book seats, make secure online payments, receive QR-code-based tickets, and interact through a real-time chat system. The platform also includes a powerful Admin Dashboard for managing movies, theaters, shows, users, and bookings.
